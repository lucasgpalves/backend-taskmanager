# Task Manager
This project is a simple management tasks, it accept the users create, update, delete and view tasks.

## Technologies
- Java
- Spring Boot
- Hibernate
- MySQL

## Main Features
- Tasks CRUD
- Keep data in MySQL
- RESTFul Interface for interaction with the tasks
